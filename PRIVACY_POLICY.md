# Privacy Policy for WhisperSnap

Effective date: May 14, 2026

WhisperSnap is a macOS menu bar application that converts speech into text. This Privacy Policy explains what data WhisperSnap processes, how that data is used, and what choices users have.

## Summary

WhisperSnap is designed to minimize data collection.

- Voice transcription can run locally on the device.
- If the user enables optional OpenAI-powered features, audio or text may be sent to OpenAI to provide those features.
- Third-party AI sharing is disabled by default and only occurs after explicit in-app consent.
- WhisperSnap does not sell personal data.
- WhisperSnap does not use third-party advertising SDKs.
- WhisperSnap does not track users across apps or websites.

## Information WhisperSnap Processes

### 1. Microphone audio

WhisperSnap requests microphone access so it can record audio for transcription.

- For local transcription, microphone audio is processed on the user's Mac.
- For optional remote realtime transcription, microphone audio may be transmitted to OpenAI or another user-configured API endpoint selected by the user.

### 2. Transcribed text

WhisperSnap processes the text generated from speech transcription.

- The text may be shown in the app interface.
- The text may be copied to the clipboard.
- In non-App Store distributions, the app may optionally attempt to insert text into the currently focused field on the Mac if the user grants Accessibility permission.
- If history is enabled, transcripts may be saved locally on the user's device.

### 3. Optional AI cleanup data

If the user enables AI Cleanup or remote transcription features, WhisperSnap may send text, audio, prompts, model identifiers, and related request metadata to OpenAI or another user-configured endpoint in order to return cleaned or realtime transcription results.

These features are optional and user-controlled. Before any third-party AI transmission, the user must enable in-app consent in Settings under "Third-Party AI Data Sharing."

### 4. API keys entered by the user

If the user enters an API key for OpenAI features, WhisperSnap stores that key locally using macOS secure storage mechanisms such as Keychain services when available.

WhisperSnap does not intentionally share the user's API key with third parties except as required to authenticate requests made by the user to the selected API provider.

## Data Collection and Storage

WhisperSnap primarily stores data locally on the user's device.

This may include:

- user settings and preferences
- selected transcription model settings
- optional transcription history
- optional API credentials stored locally

WhisperSnap does not require account creation.

## Private Mode

WhisperSnap includes an optional Private Mode setting. When Private Mode is enabled, the app is intended not to save transcription history locally.

Private Mode does not prevent network transmission when the user intentionally enables remote AI or realtime features.

## App Store Version and Direct Version

WhisperSnap may be distributed in different variants.

- The Mac App Store version uses sandbox-compatible behavior and delivers transcripts through local UI and clipboard features.
- Non-App Store distributions may offer optional direct text insertion into other apps through macOS Accessibility features if the user explicitly grants permission.

## Third-Party Services

If the user enables remote AI features, WhisperSnap may interact with:

- OpenAI
- another API provider or endpoint explicitly configured by the user

Those services process data under their own terms and privacy policies. Users should review the privacy policy of any API provider they choose to use with WhisperSnap.

WhisperSnap is intended to be used only with third-party providers that offer the same or stronger protection of user data as described in this policy.

## Tracking and Advertising

WhisperSnap does not use tracking for advertising purposes.

WhisperSnap does not use data brokers, ad networks, or cross-app tracking technologies.

## Data Sharing

WhisperSnap does not sell personal information.

WhisperSnap only shares data in the following limited cases:

- after the user grants explicit in-app consent for Third-Party AI Data Sharing and enables an optional remote AI feature
- when the user enables a remote AI or transcription provider
- when required by law, regulation, or valid legal process
- when necessary to protect the rights, safety, or integrity of the app or its users

## Data Retention

Data is retained as follows:

- local settings remain on the user's device until changed or removed
- optional transcription history remains on the user's device until deleted by the user
- remote provider data retention is governed by the privacy policy of the provider selected by the user

## User Choices

Users can control WhisperSnap's privacy behavior by:

- denying microphone permission
- leaving remote AI features disabled
- disabling Third-Party AI Data Sharing consent
- not entering an API key
- enabling Private Mode
- disabling transcription history
- removing saved API keys
- uninstalling the app and deleting its local data

## Children's Privacy

WhisperSnap is not directed to children under 13, and the developer does not knowingly collect personal information from children through the app.

## International Users

If a user enables remote AI features, their data may be transmitted to servers operated by the selected provider, which may be located outside the user's country or region.

## Changes to This Privacy Policy

This Privacy Policy may be updated from time to time. Material changes will be reflected by updating the effective date above.
